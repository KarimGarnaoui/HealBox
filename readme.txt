Voici la liste des fichiers disponibles pour le projet HealBox:

- Le pdf SUITES-HealBox-PPE1814 qui indique un possible futur pour notre projet

- Le pdf INNOVATIONOUVERTE-HealBox-PPE1814 qui résume notre synthèse de valorisation
